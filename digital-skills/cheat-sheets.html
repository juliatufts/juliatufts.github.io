Cheat Sheets

HTML

Layout

Element About Example
<!DOCTYPE html> Not an HTML element, but must be the first thing in your HTML document. <!DOCTYPE html>
<!-- All the HTML -->
<html> The first HTML element <html lang="en">
 <!-- The rest of the HTML -->
 </html>
<head> Where your metadata goes, like title and links to stylesheets <head>
 <meta charset="utf-8">
 <title>Your Title</title>
 <link rel="stylesheet" href="css/style.css">
</head>
<body> Where the content of your page goes <body>
 <!-- All page content goes here -->
</body>
<header> The top section of a page <header>
 <!-- Header content -->
</header>
<footer> The bottom section of a page <footer>
 <!-- Footer content -->
</footer>
<nav> The navigation/menu to go from page to page or section to section <nav>
 <!-- Navigation links -->
</nav>
<section> Any section within a page. Should be a sibling of header and footer <section>
 <!-- Section content -->
</section>
<div> Generic grouping element <div>
 <!-- Div content -->
</div>

Text

Element About Example
<p> A paragraph <p>
 <!-- A big long paragraph -->
</p>
<a> A link. Include href attribute. <a href="https://www.google.ca">Link to Google</a>
<a href="#id-name">Link to spot on page</a>
<ul> An unordered list (bullet points) <ul>
 <li>An item</li>
 <li>Another item</li>
</ul>
<ol> An ordered list (numbered) <ol>
 <li>First item</li>
 <li>Second item</li>
</ol>
<li> A list item. Use as children of ul or ol <li>List item</li>

Media

Element About Example
<img> An image. Include src and alt attribute <img src="image.jpg" alt="description of image">

Spacing

Element About Example
<br> A line break (like enter/return key) <p>
 <!-- First half of paragraph with a break in the middle -->
 <br>
 <!-- Second half of paragraph with a break in the middle -->
</p>
<hr> A horizontal line <p>
 <!-- A paragraph -->
</p>
<hr>
<p>
 <!-- Another paragraph that has nothing to do with the first paragraph -->
</p>
